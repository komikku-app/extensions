Tachiyomi Extensions

Guide

One click install:

Add to Mihon/Tachiyomi

If the button doesn't work, go to Settings → Browse → Extension repos and manually add https://raw.githubusercontent.com/komikku-app/extensions/repo/index.min.json

TachiyomiAZ:

  1. Go to Settings → Browse
  2. Tap on "Edit repos" and then "Add" button at bottom
  3. Input komikku-app/extensions
  4. Enjoy!

If you're not using any of the above, download and update extensions from the listing page.
